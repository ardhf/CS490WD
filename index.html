Projects

  • LED Message Board

    Developed an Internet of Things (IoT) project utilizing ESP32 microcontrollers to manage two 8x32 LED matrix displays. The system uses a web app built on PHP with a MySQL database backend. The web interface, designed with HTML forms, maintains real-time control of the LED matrices by updating the database. The ESP32s retrieve and apply the latest settings from the database asynchronously, enabling seamless functions such as message display between boards, time and weather updates, and customizable text color, background color, and brightness settings.

  • LED Light Controller

    Created a program to control LED lights using an ESP32 and three strips of WS2812 LED lights. The ESP32 retrieves the settings from a database, which is managed using PHP along with a MySQL backend. The settings allow for controlling the lights based on sunrise and sunset, setting up custom animations, adjusting brightness, and using an HTML color picker to choose any color.
